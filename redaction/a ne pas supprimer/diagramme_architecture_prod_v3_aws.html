Légende des Composants

🔶 Reverse Proxy Nginx : Point d'entrée HTTPS, load balancing, terminaison SSL
🛡️ Zone 4 Sécurité 4 middlewares : WAF, JWT Auth, RBAC, Anomaly Detection (Isolation Forest)
⚙️ APIs FastAPI 2 conteneurs (baseline + secured), Uvicorn ASGI, limites ressources 2 CPU / 4GB RAM
🤖 Modèles ML MobileNetV2 baseline (non sécurisé) vs secured (adversarial training + AES-256 + RSA-4096)
📊 Zone 5 Audit Prometheus (métriques), Grafana (dashboards), Audit Logger (hash chain immuable)
💾 Stockage PostgreSQL (users, logs, WAF) + Redis (sessions JWT, cache, rate limiting)

Flux de Requête Complet (7 étapes)

  1. Client → Nginx HTTPS (port 443, certificat TLS 1.3)
  2. WAF vérifie rate limit (100 req/min/IP via Redis) et patterns SQL injection/XSS
  3. JWT Auth valide access token RS256 (15 min expiration), session Redis
  4. RBAC vérifie permissions selon rôle (guest/agent/admin)
  5. Anomaly Detector calcule risk score (Isolation Forest), seuil 0.7
  6. API FastAPI route vers baseline (:8001) ou secured (:8002) → Modèle IA inférence
  7. Audit Logger écrit log PostgreSQL (hash chain) + Prometheus collecte métriques → Grafana dashboard

⚡ Performance : Latence P95 < 200ms (sans GPU), Throughput 100 req/sec avec 2 instances API, Uptime 99.9% avec restart automatique
