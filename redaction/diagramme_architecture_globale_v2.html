🎨 Plateformes pour modifier ce diagramme

Option 1 : Draw.io (Recommandé) ✅

  • En ligne : app.diagrams.net
  • Desktop : Télécharger depuis drawio.com
  • Avantages : Gratuit, interface intuitive, export PNG/SVG/PDF
  • Utilisation : Importer le fichier architecture_globale.drawio (à créer)

Option 2 : Excalidraw (Simple et rapide) ✅

  • En ligne : excalidraw.com
  • Avantages : Style hand-drawn, très simple, export PNG/SVG

Option 3 : Lucidchart (Professionnel)

  • En ligne : lucidchart.com
  • Avantages : Templates professionnels, collaboration
  • Inconvénient : Version gratuite limitée

Option 4 : Modifier le code Mermaid directement

  • Éditeur en ligne : mermaid.live
  • Avantages : Modification rapide du code, aperçu instantané
  • Code à modifier : Dans cette page HTML, section <div class="mermaid">

📋 Instructions pour inclure dans le mémoire LaTeX

  1. Exporter le diagramme :
    • Cliquez sur "Télécharger en PNG (haute qualité)"
    • Le fichier sera sauvegardé automatiquement
    • Renommez-le en architecture_globale.png
    • Placez-le dans figures/
  2. Dans votre fichier LaTeX :
