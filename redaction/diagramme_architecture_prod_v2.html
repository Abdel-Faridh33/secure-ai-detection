Flux de données simplifié :

  1. Utilisateur envoie requête HTTPS → Nginx (port 443)
  2. Zone 4 Sécurisation : WAF (rate limit) → JWT (authentification) → RBAC (permissions) → Anomaly Detection (scoring)
  3. API FastAPI route vers Baseline (:8001) ou Secured (:8002)
  4. Modèle IA effectue l'inférence (déchiffrement si secured)
  5. Zone 5 Audit : Logs PostgreSQL (hash chain) + Métriques Prometheus → Grafana
  6. Stockage : PostgreSQL (audit, users) + Redis (sessions, cache)
  7. Réponse JSON retournée à l'utilisateur via Nginx
